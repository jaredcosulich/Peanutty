Create A Level Build a level and share it with the community!

Here's the best way to create a level right now:

  • Find a level that you like.
  • Edit it (by clicking the 'level tab' and playing with the code) to create a new level of your own creation.
  • Use ideas and elements from other levels and combine them in to something new.
  • When your level is ready, copy and paste it in to a pastie so you can share it.
  • Let the community and me know about it by tweeting to me (@jaredcosulich) so I can post it here or by using the hashtag #peanutty.

Coding Peanutty! teaches you how to code.

Peanutty! makes it very easy for you to start coding and creating interesting things.

The language you are writing is CoffeeScript which is then compiled in to JavaScript.

Both CoffeeScript and JavaScript are languages that companies use every day to create websites and applications that are used by millions of people every day. They are very practical languages to learn if you ever want to become a professional programmer.

Even if you don't become a professional programmer, CoffeeScript and JavaScript can help you create interactive websites for what ever you can dream up. Create a blog with scrolling photos, or a map of all the places you've been. Once you understand how to program with CoffeeScript or JavaScript you can build almost anything on the Internet.

About Peanutty! What is this, who created it?

Peanutty! was created by Jared Cosulich as a way to make it easier for people to learn how to program while having fun.

Basically it works like this:

  • Person discovers Peanutty! and starts solving puzzles, conquering challenges, and just having fun.
  • Person notices that everything they do creates code in the coding area.
  • Person starts to play with the code, making larger and larger tweaks to see what happens.
  • Eventually person starts writing code directly, potentially even creating their own challenge or puzzle to share with others.
  • The next person who comes along has access to more interesting puzzles and challenges to play with.
  • Yay Peanutty!

Peanutty! is open source (warning - the code is currently an untested mess):

  • Peanutty! on GitHub.

... and was built using a bunch of great open source projects:

  • A whole bunch of Andrew McCollum's projects.
  • Box 2d JS by Erin Catto for the physics engine (there are a lot of version of Box 2d out there - this is the one I'm using right now: Box 2d Web).
  • The ACE Editor for the code editing.
  • Bootstrap for the design.

If you'd like to contact Jared or have any questions about Peanutty! you can find him on Twitter: @jaredcosulich
